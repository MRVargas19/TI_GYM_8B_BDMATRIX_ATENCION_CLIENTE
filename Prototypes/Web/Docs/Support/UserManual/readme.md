# 📘 **UserManual**
La carpeta **UserManual** contiene todos los documentos relacionados con el manual del usuario del sistema. Estos archivos proporcionan guías detalladas sobre cómo utilizar el sistema, incluyendo las funcionalidades, interfaces y procesos que los usuarios deben conocer para interactuar con el sistema de manera efectiva.
