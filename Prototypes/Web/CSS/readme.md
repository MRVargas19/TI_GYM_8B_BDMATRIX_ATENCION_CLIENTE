# CSS ![CSS](https://img.shields.io/badge/CSS-239120?&style=for-the-badge&logo=css3&logoColor=white)
La carpeta CSS contiene todas las hojas de estilo utilizadas en la interfaz web del proyecto. Aquí se encuentran los archivos .css que definen la apariencia y el diseño del sistema, incluyendo estilos generales, responsivos y específicos de cada componente. Mantener esta carpeta organizada facilita el mantenimiento y escalabilidad del diseño.
